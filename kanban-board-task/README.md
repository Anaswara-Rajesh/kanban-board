# Kanban Board
A drag-and-drop Kanban board built with React + TypeScript.

## Features
- Three columns: To Do, In Progress, Done
- Create, edit, delete tasks
- Drag and drop between columns
- localStorage 
- Clean, responsive UI

## Setup
# Clone the repository
git clone https://github.com/Anaswara-Rajesh/kanban-board.git

# Navigate into the project directory
cd kanban-board-task

# Install dependencies
npm install

# Start the development server
npm start

## Tech Stack
- React 18
- TypeScript
- Native Drag & Drop API
- localStorage
